软件包管理

  • rpm是一个中软件包管理，同时一种软件包格式

RPM包管理器

RPM的软件包查询

  1. rpm -qa 查询已经安装的软件名称

  2. rpm -q <软件包> 查询软件是否安装，安装过则输出该软件包的信息

RPM的软件包的下载

yum -ivh <软件包>

i：表示安装

v：表示更详细的安装信息页面

h：表示显示安装进度

例如;从镜像网站下载一个rpm包
  1. 可以先使用wget命令下载，wget -O （表示下下载到指定的目录，如果没有-O，则是下载到当前目录）

yum软件包管理

  • 相关命令：Linux uname命令用于显示系统信息，uname可显示电脑以及操作系统的相关信息

软件的类型

  1. 源码包 需要编译
  2. 二进制包 已编译
  3. 在ubantu系统上，在线包管理工具是apt,而在红帽中则为yum（在线工具）管理

使用yum管理RPM包

安装
  1. yum -y install <包名>
  2. yum -y reinstall <包名>
  3. yum -y update <包名>
  4. yum -y update 更新所有包
卸载
  1. yum remove <包名>
查询

  1. yum list <包名>

  2. yum info <包名> 可以查询包的信息（无论是否安装过）

  3. yum history info 2 、yum history

  4. 撤销yum history的动作，yum history undo

  5. yum search <包名> 搜索软件包

  6. yum provides <文件> 查询命令提供的软件包

SELinux基础

SELinux可以给文件，目录，端口，进程打上标签，标签称为上下文

不同的标签就不能访问各自的上下文

httpd服务的进程上下文：httpd_t

httpd服务的文件上下文：httpd_sys_content_t

httpd服务的进程上下文：http_port_t

模式

  1. enforcing （强制模式）启动SELinux，禁用违反策略的行为，并发送警告
  2. permissive（允许模式）启用SELinux，不禁用违反策略的行为
  3. disabled（禁用模式）

临时更改：setenforce 1 | 0

永久更改：/etc/selinux/config

文件上下文

查看文件上下文

ls -Z[d] 查看目录或文件的上下文 -Z 表示查看上下文，d表示查看文件，目录本身

添加文件上下文

semanage fcontext -a -t <上下文> <目录文件> a表示添加，-t表示上下文类型

restorecon -RFvv <目录文件> vv表示查看过程

修改文件上下文

semanage fcontext -m -t <上下文> <目录文件> m表示修改，-t表示上下文类型

restorecon -RFvv <目录文件> vv表示查看过程

临时修改上下文：chcon -t <上下文类型> <文件目录>，内核安全上下文列表（semanage fcontent -l）经过刷新后（restorecon -RFvv < 文件>）就会恢复之前的上下文

命令：
  1. chcon -t <上下文类型> <文件、目录> 临时修改
  2. semanage fcontent -a -t <上下文类型> <文件、目录> 添加默认上下文
  3. semanage fcontent -m -t <上下文类型> <文件、目录> 修改默认上下文
  4. restorecon -RFvv <文件。目录> 当这个文件在策略里有定义是，可以恢复原来的 文件标签。

总结：

文件或者目录要写成绝对路径，记得查看文件安全上下文列表，restorecon 就是重新加载这个列表

此外，要就注意移动文件和复制文件对于上下文的不同影响

端口上下文

查看
添加端口上下文

semanage port -a -t <上下文> -p <tcp | udp> <端口>

例如：semanage port -a -t httpd_port_t -p tcp 10000

删除端口上下文

semanage port -d -t <上下文> -p <tcp | udp> <端口>

例如：semanage port -d -t httpd_port_t -p tcp 10000

修改端口上下文

semanage port -m -t <上下文> -p <tcp | udp> <端口>

例如：semanage port -m -t httpd_port_t -p tcp 10000

布尔值

布尔值是selinux的策略开关

查看布尔值

getsebool -a

修改布尔值

setsebool -P （P表示永久修改）

例如：setsebool -P nfs_export_all_rw off

查看用户修改的布尔值

semanage boolean -l -C

LDAP

用户集中管理，可以理解为将所有用户的信息存放在一个特定的服务器中，然后用户的信息存放在不同的目录下，这样我们可以在本机连接LDAP服务器，就可以共享该用户的信息

连接LDAP服务器

  1. 安装软件包 yum -y install authconfig-gtk sssd

  2. 执行authconfig-gtk

  3. 选择LDAP认证

  4. 输入域名 dc=xxx,dc=xxx

  5. 输入LDAP服务器的ip地址或主机名

  6. √ TLS，下载证书

  7. 选择LDAP password

  8. 连接成功后，可以查看服务器内的用户数据

  9. 查看用户 id <用户> （该用户不是本地创建的，cat/etc/passwd查看）
