Java-JSP小项目部署

发表于 2020-07-16 分类于 Java 评论数： 阅读次数：
本文字数： 4k 阅读时长 ≈ 4 分钟
查看用过yum安装的java环境
删除linux上的java
安装java，创建/usr/mysoftware/java目录，修改权限，chmod 777 /usr/mysoftwart/java；
下载 java，通过工具上传到/usr/mysoftwart/java目录下，解压；
编辑配置文件 /etc/profile， source /etc/profile更新
验证：java，javac，java -version，echo $PATH，echo $CLASSPATH
安装Tomcat（官网下载压缩包），解压即可
启动，关闭
打包mysql数据库
云主机的mysql创建数据，然后根据刚刚打包生成compny.txt创建数据表
创建成功
打包项目：IDEA把项目打包成war包，将.war文件上传到Tomcat的webapps的目录即可，自动生成项目。

然后就可以访问啦！🎈🎈🎈🎈🎈🎈
